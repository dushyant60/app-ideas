# Application Name: Task List

**Tier:** 1-Beginner

**Purpose:**

The purpose of this application is to help users keep track of their tasks and to-do lists. It will provide a simple and user-friendly interface for users to add, update, and mark tasks as completed.

**Resources Needed:**

- Basic knowledge of programming fundamentals.
- A programming environment (e.g., IDE like Visual Studio Code, or an online code editor).
- Familiarity with a programming language such as JavaScript.

## User Stories

- [ ] User can view a list of tasks.
- [ ] User can add a new task to the list.
- [ ] User can mark a task as completed.
- [ ] User can edit the task description.
- [ ] User can delete a task.
- [ ] User can clear all completed tasks.

## Bonus Features

- [ ] User can categorize tasks into different lists (e.g., work, personal, shopping).
- [ ] User can set due dates for tasks.
- [ ] User can prioritize tasks (e.g., high, medium, low).
- [ ] User can search for tasks by keyword.
- [ ] User can set reminders or notifications for important tasks.
